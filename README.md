Задание 2.

Инструкция по запуску автоматических тестов

1. Необходимо чтобы был установлен python
2. В папке с проектом открыть консоль
3. Создать виртуальное окружение с помощью команды в консоле python -m venv venv
4. Активировать виртуальное окружение командой venv\Scripts\activate
5. Установить необходимый набор пакетов pip install -r requirements.txt.
6. Запустить автоматизированные тесты командой pytest (может запуститься не сразу)

Команды для запуска:

pytest - запуск всех тестов

pytest tests\test_statistics.py - запуск тестов получения статистики

pytest tests\test_get_item.py - запуск тестов получения объявления по id айтема

pytest tests\test_seller_items.py - запуск тестов получения объявлений по id продавца

pytest tests\test_create_item.py - запуск тестов создания объявления


В проекте используются инструменты для автоматической проверки и форматирования кода (линтер и форматтер):

ruff — выполняет статический анализ кода, находит ошибки, потенциальные баги и проблемы со стилем  
black — автоматически форматирует код в едином стиле  
isort — упорядочивает импорты  

Настройки инструментов заданы в pyproject.toml и включают:

единый лимит длины строки (88 символов)  
совместимость с Python 3.11  
использование стандартов PEP8 для проверки кода  
автоматическую сортировку импортов  
отключение проверки длины строки (так как этим занимается black)  

Проверка кода без внесения изменений:

ruff check .  
black --check .  
isort --check-only .  

Применение исправлений:

ruff check . --fix   
isort .  
black .  

Скриншоты Allure-отчета находятся в файле Allure.docx

Команды для генерации allure-отчета (для этого необходим allure):  
pytest --alluredir=allure-results  
allure serve allure-results
